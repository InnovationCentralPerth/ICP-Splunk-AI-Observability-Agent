1
Splunk Real-Time Dashboard
Operational visibility across the ICP space
  • Live zone occupancy from MV cameras — people count per defined area
  • Environmental conditions panel — temperature & humidity from MT sensors
  • Network activity view — connected device count and switch telemetry
  • All data delivered via Splunk HEC as structured JSON events (no agents, no forwarders)
  • Splunk AI (MLTK) runs anomalydetection and predict/LLP5 on ingested data — results fed back into LLM context
  • Immediately deployable on any Splunk Enterprise instance — no custom app required
2
Agentic AI Observability App + Automated Response Engine
Closed-loop: Splunk AI detects → LLM interprets → Splunk records the response
  • Ask questions in plain English — responses always explicitly cite Splunk AI findings
  • Automated AI Response Engine — when an anomaly is detected, the LLM auto-generates an incident report, creates a Splunk saved alert via REST API, and logs icp:automated_response back to Splunk
  • Two detection tiers: Splunk MLTK statistical outliers (primary) + threshold breaches (secondary)
  • Live in-app LLM switching — GPT-4o mini (analyst) · Llama 3.3 70B (chat) · Gemini Flash 2.0 (fallback)
  • Clickable 12-hour historical trend charts pre-populated from Splunk time-series data
  • Extensible: any OpenRouter-compatible model swappable via single env variable
